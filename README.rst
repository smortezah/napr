All the tutorials are presented `here <https://github.com/smortezah/napr/tree/main/tutorials>`_.

Testing
~~~~~~~

You can launch the test suite, after installation (``pytest`` >= |PyTestMinVersion| is required):

.. code-block:: console 

    pytest sklearn
